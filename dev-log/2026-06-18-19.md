### 5. 책장 페이지 (`src/pages/bookshelf/page.tsx` + `src/mocks/bookshelf.ts`)

**변경 사항**
- `RECENT_STORIES` mock 데이터를 홈 화면(`recentStories`)과 동일한 6개 동화로 교체
  - 도깨비 방망이와 지우의 모험 / 한옥 마을의 작은 호랑이 친구 / 설날 밤 별빛 떡국 이야기
  - 제주 바람과 작은 해녀의 노래 / 탈춤 추는 토끼 달이 / 한복 입은 아기 별의 첫 여행
- `ALL_STORIES`도 동일 6개로 통일, 각 동화에 `tag` · `status(reading/completed)` 부여
- `PREMIUM_RECENT_STORIES` = `RECENT_STORIES` alias로 유지 (premium 페이지 호환)
- 최근에 읽은 동화 카드 그리드: `grid-cols-3` → `grid-cols-2 md:grid-cols-3` (6개 전체 표시)
- 이미지 위에 태그 뱃지 오버레이 추가
- 저장된 동화 테이블: `태그` 컬럼 추가, `slice(0, 3)` 제거 → 필터 결과 전체 표시
- "총 N권" 동적 카운트 (`ALL_STORIES.length`)
- 이어 읽기/다시 읽기 버튼 → `Link` 컴포넌트로 교체 (`/viewer/:id` 라우팅)
